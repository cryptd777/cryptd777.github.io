Proxy manager

ProxyMaster

Linux GUI application for managing system proxy settings and package manager proxy configuration.

Python PyQt Linux

Latest release: v2.2 (Sep 6, 2025)

Key features

  • Edit and save proxy environment variables
  • Toggle proxy ON/OFF for system and shell
  • Manage apt, dnf, pacman, zypper
  • History of successful proxy values

Requirements

  • Python 3.7+
  • PyQt6 (or PyQt5 fallback)
  • pkexec for system config changes

Source code, releases, and AppImage live on GitHub.
