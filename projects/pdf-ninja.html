PDF Ninja

Fast, safe PDF workflows for Linux. Everything runs locally with no uploads.

Linux Offline PDF AppImage
v0.0.1 Releases GitHub

Snapshot

  • Platform: Linux
  • UI: CustomTkinter (Python)
  • Focus: Local, offline PDF workflows
  • Latest release: v0.0.1
  • Bundled tools: pdfcpu, mutool, qpdf, ghostscript, img2pdf, poppler, ocrmypdf

Key features

  • Merge, split, compress, convert, extract
  • Protect and repair PDF files
  • Drag-and-drop friendly UI
  • No uploads: all processing stays local

Supported tasks

Merge, split, compress, convert, extract, protect, info, and repair.

Build targets

Standalone build, DEB package, and AppImage options.

Safe by default

PDFs never leave your machine, keeping sensitive documents private.
